# 数据清理和初始化系统实现

## 概述

为 ManualBox 应用实现了全面的数据清理和初始化系统，确保数据结构干净且无重复，提供了强大的数据管理功能。

## 实现的功能

### 1. 数据诊断服务 (DataDiagnostics.swift)

**增强的诊断功能：**
- 检测重复分类和标签
- 统计所有实体数量（产品、订单、说明书、维修记录）
- 识别孤立数据（无关联的订单、说明书）
- 检测空实体（无产品的分类/标签）
- 检查初始化状态
- 生成详细的诊断报告

**新增诊断项目：**
- 无分类的产品
- 孤立的订单和说明书
- 空分类和标签
- 数据完整性检查

### 2. 数据清理服务 (DataCleanupService.swift)

**全面的清理功能：**
- 清理重复分类（保留较早创建的，转移产品关联）
- 清理重复标签（保留较早创建的，转移产品关联）
- 删除孤立的订单和说明书
- 清理空实体（保留默认分类和标签）
- 完整数据重置功能

**清理策略：**
- 智能合并：保留较早创建的实体，转移所有关联
- 安全删除：只删除非默认的空实体
- 关系维护：确保数据关系完整性

### 3. 数据初始化服务 (DataInitializationService.swift)

**智能初始化：**
- 检测首次启动
- 版本迁移支持
- 条件性数据创建（只在需要时创建）
- 示例数据管理

**初始化策略：**
- 首次启动检测：使用 UserDefaults 标记
- 版本追踪：记录最后初始化的应用版本
- 条件创建：只在数据表为空时创建默认数据
- 可选示例数据：可控制是否创建示例产品

### 4. 数据验证服务 (DataValidationService.swift)

**实时验证：**
- 分类名称唯一性验证
- 标签名称唯一性验证
- 产品数据完整性验证
- 关系完整性验证

**验证类型：**
- 错误：重复名称、缺失必需字段、无效关系、数据损坏
- 警告：空实体、未使用实体、潜在重复、缺失可选数据

### 5. 数据管理界面 (DataManagementView.swift)

**用户友好的管理界面：**
- 数据统计显示
- 一键诊断功能
- 数据验证工具
- 自动修复选项
- 完整清理功能
- 危险操作确认

**界面功能：**
- 实时数据统计
- 详细诊断报告查看
- 验证结果展示
- 进度指示器
- 操作结果反馈

## 应用启动流程优化

### 新的启动序列：

1. **数据诊断** - 检查当前数据状态
2. **问题修复** - 如发现问题，自动修复
3. **智能初始化** - 根据需要创建默认数据
4. **数据验证** - 最终验证数据完整性
5. **通知注册** - 完成其他初始化任务

### 启动日志：
```
[App] 开始应用初始化...
[App] 数据诊断结果: 数据状态良好，未发现问题
[App] 初始化结果: 非首次启动，跳过初始化
[App] 数据验证结果: 数据验证通过
[App] 应用初始化完成
```

## 内存管理

### 防止重复数据的机制：

1. **创建时验证** - 在创建新实体前检查重复
2. **导入时检查** - 数据导入时自动去重
3. **定期清理** - 提供手动清理工具
4. **实时监控** - 持续监控数据完整性

### 数据一致性保证：

- 事务性操作：所有清理操作在事务中执行
- 关系维护：删除实体时正确处理关联关系
- 错误恢复：操作失败时回滚更改
- 日志记录：详细记录所有操作
