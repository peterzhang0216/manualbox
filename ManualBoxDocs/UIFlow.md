# 🎨 ManualBox UI 页面流程与设计指南

## 📱 整体设计理念

ManualBox 采用现代化的多平台设计理念，提供一致且直观的用户体验：

- **简洁优先**：界面简洁明了，减少认知负担
- **多平台适配**：iOS 和 macOS 原生体验
- **响应式设计**：适配不同屏幕尺寸
- **无障碍友好**：支持 VoiceOver 和键盘导航

## 🏗️ 应用架构流程

### 主界面结构

```
iOS (TabView)                    macOS (NavigationSplitView)
├── 产品管理 Tab                  ├── 侧边栏
├── 分类管理 Tab         →       ├── 内容区域  
├── 标签管理 Tab                  └── 详情面板
└── 设置 Tab
```

### 导航层级

```
Root
├── 产品管理
│   ├── 产品列表 → 产品详情 → 编辑产品
│   ├── 添加产品
│   └── 搜索筛选
├── 分类管理
│   ├── 分类列表 → 分类详情 → 编辑分类
│   └── 添加分类
├── 标签管理
│   ├── 标签列表 → 标签详情 → 编辑标签
│   └── 添加标签
└── 设置
    ├── 数据同步
    ├── 导入导出
    └── 应用信息
```

## 🎨 UI 组件设计规范

### 颜色系统
- **主色调**：系统蓝色 (AccentColor)
- **辅助色**：系统灰色层级
- **标签色**：彩虹色系，提供视觉区分
- **状态色**：成功(绿)、警告(橙)、错误(红)

### 字体规范
- **标题**：Large Title, Title, Title2
- **正文**：Body, Callout
- **辅助**：Caption, Caption2

### 间距系统
- **基础单位**：8pt
- **组件间距**：16pt
- **内容边距**：20pt
- **卡片圆角**：12pt

## 📱 核心页面流程

### 1. 产品管理流程

#### 产品列表页
```
功能布局：
├── 搜索栏 (支持名称、品牌、型号搜索)
├── 筛选器 (分类、标签、状态)
├── 排序选项 (名称、更新时间、创建时间)
└── 产品网格/列表
    ├── 产品图片
    ├── 产品名称
    ├── 品牌型号
    └── 标签显示
```

**交互行为：**
- 下拉刷新数据
- 无限滚动加载
- 手势操作：左滑删除(iOS)，右键菜单(macOS)
- 多选批量操作

#### 产品详情页
```
信息层级：
├── 产品头图
├── 基本信息
│   ├── 名称、品牌、型号
│   ├── 分类和标签
│   └── 创建/更新时间
├── 说明书管理
│   ├── PDF 文档列表
│   ├── 在线查看
│   └── 下载管理
├── 维修记录
│   ├── 记录列表
│   ├── 费用统计
│   └── 添加记录
└── 操作按钮
    ├── 编辑产品
    ├── 分享产品
    └── 删除产品
```

**交互行为：**
- 图片预览和编辑
- PDF 文档在线查看
- 维修记录时间线展示
- 数据导出分享

## 🔄 用户交互设计

### 手势操作 (iOS)
- **下拉刷新**：更新数据
- **左滑**：快速删除
- **长按**：进入多选模式
- **双击**：快速编辑
- **捏合**：缩放图片

### 键盘快捷键 (macOS)
- **⌘N**：新建项目
- **⌘E**：编辑当前项目
- **⌘F**：搜索
- **⌘⌫**：删除选中项目
- **空格**：快速预览

### 上下文菜单
```
产品项目右键菜单：
├── 编辑产品
├── 复制产品
├── 移动到分类
├── 添加标签
├── 导出数据
└── 删除产品
```

## 📊 状态管理和反馈

### 加载状态
- **骨架屏**：首次加载时显示
- **进度指示器**：数据加载过程
- **刷新动画**：下拉刷新反馈
- **空状态**：无数据时的友好提示

### 错误处理
- **网络错误**：显示重试选项
- **数据错误**：显示错误信息和解决方案
- **权限错误**：引导用户授权
- **同步错误**：显示同步状态和手动同步选项

### 成功反馈
- **Toast 提示**：操作成功的轻量反馈
- **进度条**：批量操作进度
- **动画效果**：删除、添加的视觉反馈

## 🎯 搜索和筛选设计

### 搜索功能
```
搜索范围：
├── 产品名称 (权重最高)
├── 品牌信息
├── 型号信息
├── 分类名称
├── 标签名称
└── 备注内容
```

### 筛选器设计
```
筛选选项：
├── 分类筛选 (单选/多选)
├── 标签筛选 (多选)
├── 时间范围 (创建时间、更新时间)
├── 状态筛选 (有说明书、无说明书)
└── 自定义排序
```

## 📱 响应式适配

### iOS 适配
- **iPhone**：单栏布局，Tab 导航
- **iPad**：双栏/三栏布局，侧边栏导航
- **紧凑模式**：适配小屏幕设备

### macOS 适配
- **三栏布局**：侧边栏 + 内容 + 详情
- **工具栏**：常用操作快速访问
- **菜单栏**：完整功能访问
- **窗口管理**：支持多窗口操作

## 🎨 动画和过渡

### 页面过渡
- **推入动画**：层级导航
- **淡入淡出**：模态弹窗
- **滑动切换**：Tab 切换
- **缩放动画**：图片预览

### 微交互
- **按钮反馈**：点击缩放效果
- **列表更新**：插入/删除动画
- **状态变化**：颜色过渡
- **加载动画**：旋转指示器

## 🌙 深色模式适配

### 颜色适配
- **自动切换**：跟随系统设置
- **手动切换**：应用内设置
- **高对比度**：无障碍支持

### 图片适配
- **动态图片**：根据模式自动切换
- **透明处理**：保持一致的视觉效果

## ♿ 无障碍设计

### VoiceOver 支持
- **语义化标签**：所有UI元素都有清晰的描述
- **导航顺序**：合理的焦点移动顺序
- **状态广播**：及时反馈操作结果

### 键盘导航
- **Tab 顺序**：逻辑清晰的Tab移动
- **快捷键**：常用操作的键盘快捷方式
- **焦点指示**：清晰的焦点视觉反馈

### 字体和对比度
- **动态字体**：支持系统字体大小设置
- **高对比度**：确保文字清晰可读
- **颜色无关**：不仅依赖颜色传达信息

## 📱 具体实现细节

### iOS 专用特性
- **3D Touch/Haptic Touch**：快速操作
- **Spotlight 搜索**：系统级搜索集成
- **Siri 快捷指令**：语音操作支持
- **Widget 小组件**：主屏幕快速访问

### macOS 专用特性
- **Touch Bar**：MacBook Pro 支持
- **菜单栏集成**：系统菜单栏快速访问
- **拖拽操作**：文件拖拽导入
- **多显示器**：扩展屏幕支持

这个 UI 流程设计确保了 ManualBox 在不同平台上都能提供优秀的用户体验，同时保持功能的一致性和易用性。
