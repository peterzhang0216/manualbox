# ManualBox

保修信息管理助手 | Warranty & Asset Manager

ManualBox 是一款专为个人和家庭设计的保修/资产/说明书管理工具，支持多平台（macOS/iOS），界面美观，交互流畅，助您轻松管理各类产品的保修、维修、资产信息与文档。

---

## ✨ 主要特性

- **多平台支持**：原生 SwiftUI，兼容 macOS 与 iOS。
- **三栏/多栏自适应布局**：主界面与产品列表支持多栏同步，体验一致。
- **产品/资产管理**：支持分类、标签、保修期、维修记录、说明书等信息管理。
- **通知与提醒**：高级推送设置，支持多渠道、静默时段、定制提醒。
- **数据导入/导出/备份**：支持本地/iCloud 数据备份与恢复，导入导出商品、分类、标签等。
- **OCR 文字识别**：添加说明书时可自动识别文字内容。
- **主题与外观**：支持浅色/深色/跟随系统主题，主题色自定义。
- **国际化**：支持中文、英文，内置语言切换，界面即时刷新。
- **隐私合规**：权限申请与 Info.plist 描述规范，隐私政策与用户协议入口。

## 🚀 安装与运行

### macOS
1. 克隆本仓库：
   ```bash
   git clone https://github.com/yourusername/ManualBox.git
   ```
2. 使用 Xcode 15+ 打开 `ManualBox.xcodeproj`。
3. 选择 `My Mac` 作为目标，编译并运行。

### iOS
1. 同上，选择 `iOS` 设备或模拟器。
2. 需 Apple Developer 账号签名。

## 🛠 功能说明

### 主界面
- 多栏自适应，支持产品、分类、标签、设置等模块切换。
- 主题切换（浅色/深色/跟随系统）与主题色自定义，全局生效。

### 产品/资产管理
- 支持商品/资产的添加、编辑、分类、标签、保修期、维修记录、说明书等信息管理。
- 支持 OCR 文字识别，自动提取说明书内容。

### 通知与提醒
- 高级推送设置，支持：
  - 保修到期提前提醒（自定义天数）
  - 维修进度推送（自动）
  - 资产定期巡检/保养（年/半年/季度）
  - 通知渠道选择（系统通知/邮件/日历事件）
  - 静默时段设置

### 数据导入/导出/备份
- 支持导入/导出商品、分类、标签等数据（开发中）。
- 支持本地/iCloud 数据备份与恢复（开发中）。

### 设置页面
- 两栏分组导航，卡片式布局，视觉风格统一。
- 主题模式切换（浅色/深色/跟随系统），即时生效。
- 主题色自定义（系统/蓝/绿/橙/粉/紫/红），带视觉反馈。
- 语言切换（跟随系统/中文/英文），即时刷新。
- 默认保修期、OCR 默认开关。
- 隐私政策、用户协议、检查更新等入口。

## 🎨 主题系统

### 主题模式
- **浅色模式**：明亮的背景和深色文字，适合日间使用。
- **深色模式**：暗色背景和浅色文字，减少屏幕光污染，适合夜间使用。
- **跟随系统**：自动跟随系统设置切换明暗模式。

### 主题色
提供多种主题色选择，全局生效：
- **系统默认**：跟随系统强调色
- **蓝色**：商务风格
- **绿色**：自然清新
- **橙色**：活力四射
- **粉色**：温馨浪漫
- **紫色**：高雅神秘
- **红色**：热情张扬

## 🌏 国际化与语言切换

- 支持中文（简体）与英文，自动跟随系统语言。
- 设置页可手动切换语言，切换后界面即时刷新（无需重启）。
- 所有界面文本均已本地化，新增文本请维护 `zh-Hans.lproj/Localizable.strings` 与 `en.lproj/Localizable.strings`。
- 权限描述已本地化，请维护 `zh-Hans.lproj/InfoPlist.strings` 与 `en.lproj/InfoPlist.strings`。

---

## 🔒 权限与隐私合规

- **通知权限**：用于推送保修/维修/巡检等提醒。
- **本地/iCloud 存储**：用于数据备份与恢复。
- **相册/文件访问**：仅在导入说明书图片/文档时请求。
- **Info.plist** 权限描述已按 Apple 设计规范补充，并支持本地化。
- 设置页提供隐私政策、用户协议入口，合规透明。

---

## 📝 Info.plist 权限描述与本地化

- 所有敏感权限均在 Info.plist 中添加了用途说明（NSPhotoLibraryUsageDescription、NSUserNotificationUsageDescription 等）。
- 权限描述已本地化，支持中英文切换。
- ATS、后台推送等配置已按 Apple 规范设置。
